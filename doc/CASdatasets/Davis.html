Davis {tools}R Documentation

Davis dataset

Description

This dataset contains information of 200 individuals.

Usage

data(Davis)

Format

data is a data frame of 5 columns and 200 rows:

sex

a factor: M for male and F for female.

weight

a numeric for the weight in Kg.

height

a numeric for the height in cm.

reportedWeight

a numeric for the weight in Kg.

reportedHeight

a numeric for the height in cm.

Source

https://socialsciences.mcmaster.ca/jfox/Books/Applied-Regression-2E/datasets/Davis.txt

References

Davis (1990) Body image and weight preoccupation: A comparison between exercising and non-exercising women, Appetite, 15, 13-21, doi:10.1016/0195-6663(90)90096-q.

Examples


# (1) load of data
#
data(Davis)
dim(Davis)
head(Davis)
