fredpt17 {tools}R Documentation

French geospatial dataset for departments

Description

fredpt17 is a spatial database containing geospatial information of French departments to be used with pricing actuarial games' files, see pricingame. fredpt17 is a geospatial dataframe of class sp based on six files: DEPARTMENTS.cpg, DEPARTMENTS.dbf, DEPARTMENTS.prj, DEPARTMENTS.qpj, DEPARTMENTS.shp, DEPARTMENTS.shx. As it is of class sp, fredpt17 can be easily plotted or summarized.

Usage

data(fredpt17)

Format

fredpt17@data contains 1 column:

DEPT

A factor for the department number.

Source

See https://actinfo.hypotheses.org/86 for the third pricing game.

See Also

See the sp class.

Examples
