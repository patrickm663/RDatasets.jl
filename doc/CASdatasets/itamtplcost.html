itamtplcost {tools}R Documentation

Italian MTPL cost

Description

This dataset contains large losses (in excess of 500 Keuro) of an Italian Motor-TPL company since 1997.

Usage

data(itamtplcost)

Format

itamtplcost is a data frame of 2 columns and 457 rows:

Date

Date of loss (accident date).

UltimateCost

Ultimate cost trended to 2013 and developed to ultimate losses.

Source

Unknown private insurer.

Examples


# (1) load of data
#
data(itamtplcost)
